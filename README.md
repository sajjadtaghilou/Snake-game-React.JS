# React-snake [Build Status](https://github.com/sajjadtaghilou/snake)]
> The classic game "_Snake_" in React

## Install

Clone this repo following command in your favorite shell:
* `cd snake`
* `npm install`
* `npm start`
* visit http://localhost:3000/

Check it out:
https://www.snake.sajjadtaghilou.com/
